# 右肺摘除

Personal translation archive site, deployed via GitHub Pages.

## Stack

- Vite 6 + React 18
- react-router-dom 6 (HashRouter for static deployment compatibility)
- Plain CSS, no UI framework
- Archive data stored as JSON files under `public/data/`, fetched at runtime

## Deploy

```bash
npm run deploy   # build and push to gh-pages branch
```
